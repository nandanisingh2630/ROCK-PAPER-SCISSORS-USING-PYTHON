**Rock Paper Scissors Game with Home Screen**

This project is a **Rock Paper Scissors Game** developed using Python with the **Tkinter GUI library** and **PIL (Python Imaging Library)** for handling images. The application begins with a **Home Screen** that welcomes the user and provides a **Start Game** button. When the user clicks the Start Game button, the home screen closes and the main game window opens.

The main game window displays graphical images for both the **user** and the **computer** choices. The images for Rock, Paper, and Scissors are loaded, resized, and displayed using PIL and Tkinter labels. The interface also includes **score counters** for the player and the computer, along with indicators showing which side belongs to the user and the computer. A message area is provided to display game results such as **You Win, You Lose, or It's a Tie**.

The user selects an option using **Rock, Paper, or Scissor buttons**. When a button is clicked, the program randomly generates the computer’s choice using the random module. The selected images are updated on the screen, and the program compares both choices using conditional statements to determine the winner. Based on the result, the message is displayed and the corresponding score is updated.

The game supports **multiple rounds**, dynamically updates scores, and provides an interactive graphical interface. This project demonstrates the use of **Tkinter widgets (Label, Button, Frame)**, **event handling**, **functions**, **conditional logic**, **random number generation**, **image handling with PIL**, and **GUI-based game development in Python**.
